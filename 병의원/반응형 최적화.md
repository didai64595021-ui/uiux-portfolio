[PART 1] 전역 CSS 설정 추가
--------------------------------
1) 내 프로젝트에서 전역으로 적용되는 CSS 파일(예: styles.css, global.css, app.css 등)을 찾아.
2) 그 파일의 맨 아래에 아래 코드를 그대로 추가해.

/* ===== 한국어 텍스트 공통 스타일 ===== */
.korean-text {
  /* 한글 단어(어절) 단위 줄바꿈: 단어 중간에서 안 끊기게 함 */
  word-break: keep-all;      /* CJK 언어용 어절 단위 줄바꿈 [참고: word-break keep-all 설명][web:47][web:38] */
  
  /* 제목/짧은 문장의 줄 길이를 균형 있게 맞춤 (지원 브라우저: 최신 Chrome, Firefox, Safari 등) */
  text-wrap: balance;        /* 줄 길이 균형 맞추기 [web:42][web:51] */
  
  /* 단락을 양쪽 정렬해서 폭을 일정하게 보이도록 함 */
  text-align: justify;
  
  /* 영어 등에 하이픈을 자동으로 넣어 남는 공간 줄이기 (필요 없으면 주석 처리 가능) */
  hyphens: auto;
  
  /* 한 줄 길이가 너무 길어지지 않게 최대 폭 제한 (원하는 값으로 조정 가능) */
  max-width: 36rem;
  margin: 0 auto;            /* 가운데 정렬 */
  line-height: 1.5;
}

/* 모든 제목에 균형 잡힌 줄바꿈 적용 (필요 없으면 h1~h3만 남겨도 됨) */
h1, h2, h3, h4, h5, h6 {
  text-wrap: balance;        /* 제목 줄 길이 균형 맞추기 [web:42] */
  word-break: keep-all;      /* 한글 제목이 단어 중간에서 안 끊기도록 함 [web:47][web:38] */
}

/* ===== 모바일(태블릿 포함) 최적화 ===== */
@media (max-width: 768px) {
  .korean-text {
    font-size: 0.95rem;      /* 모바일에서 글자가 너무 크지 않게 조정 */
    line-height: 1.6;        /* 모바일에서 조금 더 넉넉한 줄 간격 */
    max-width: 90%;          /* 좌우 여백 확보 */
  }
}


[PART 2] 솔루션 섹션 HTML/JSX에 클래스 적용
---------------------------------------------
1) “프리미엄 온라인 경험 & 데이터 기반 마케팅” 문구가 들어있는 섹션을 찾아.
2) 그 텍스트 전체를 감싸는 요소(div, section 등)에 .korean-text 클래스를 붙여.

(예시 1: 순수 HTML)

<section class="solution">
  <div class="korean-text">
    <h2>프리미엄 온라인 경험 &amp; 데이터 기반 마케팅</h2>
    <p>
      "백화점에서 느끼는 럭셔리한 경험을 스크린 안에서도 동일하게 전달한다"는 핵심 원칙 아래,
      프리미엄 이커머스 플랫폼과 퍼포먼스 마케팅 전략을 동시에 설계했습니다.
    </p>
  </div>
</section>

(예시 2: React / Next.js JSX)

<section className="solution">

      프리미엄 이커머스 플랫폼과 퍼포먼스 마케팅 전략을 동시에 설계했습니다.

[PART 3] 다른 큰 제목/설명에도 재사용
---------------------------------------
1) 사진 1처럼 줄바꿈/정렬을 예쁘게 맞추고 싶은 모든 큰 제목과 핵심 카피 영역을 찾아.
2) 각 텍스트 블록을 감싸는 요소에 .korean-text를 추가해.

(예시)

<div class="hero-title korean-text">
  <h1>당신의 브랜드를 위한 프리미엄 온라인 경험</h1>
</div>

<article class="case-detail">
  <div class="korean-text">
    <p>데이터 기반 마케팅 전략으로 전환율을 극대화했습니다.</p>
  </div>
</article>


[PART 4] 줄바꿈 위치를 더 세밀하게 지정하고 싶을 때 (선택)
--------------------------------------------------------------
1) 어떤 기기에서 보더라도 특정 위치에서 줄바꿈이 되도록 “우선候補”를 주고 싶다면 <wbr> 태그를 사용해.
2) 클로드에게 “아래 예시처럼 줄이 바뀌었으면 하는 단어 사이에 <wbr>을 넣어 달라”고 시켜.

(예시)

<h2 class="korean-text">
  프리미엄 온라인 경험<wbr> &amp; 데이터 기반 마케팅
</h2>

<p class="korean-text">
  "백화점에서 느끼는 럭셔리한 경험을<wbr> 스크린 안에서도 동일하게 전달한다"는<wbr>
  핵심 원칙 아래, 프리미엄 이커머스 플랫폼과 퍼포먼스<wbr> 마케팅 전략을 동시에 설계했습니다.
</p>

※ 설명: <wbr>은 “여기서 줄을 바꿀 수 있다”는 힌트를 주는 태그라서, 브라우저가 줄바꿈이 필요할 때 우선적으로 그 위치에서 줄을 바꾸려고 한다.[web:46][web:49]

  줄바꿈을 더 세밀하게 제어해야 하는 문장은 [PART 4]처럼 <wbr>을 사용해서 수정해 줘.”[web:42][web:51][web:38]
